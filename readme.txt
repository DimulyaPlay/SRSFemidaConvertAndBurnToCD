Программа для сбора аудиозаписей судебных заседаний из произвольного количества источников(Shared folders),
конвертации в формат MP3, прослушивания и записи на CD/DVD диски.

Предположительный функционал

1 Параметры
- добавление/удаление залов судебных заседаний(источников)
- настройка для каждого зала при обнаружении новых записей - автоконвертация, стенографирование записей (низкокачественное)
- частота сканирования на наличие новых записей
- расписание сканирования

2 Поиск
- просмотр записей списками по залам
- поиск по номеру дела, дате/периоду, длительности

3 Основные
- запись на CD/DVD
- прослушивание аудиозаписи
- регулировка скорости
- регулировка громкости

Минусы реализации
- необходимо наличие минимум windows 7 c .net 4.7.2+
